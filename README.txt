# Análise de Bolsas ProUni - Pernambuco

Este projeto analisa a distribuição de bolsas do ProUni no estado de Pernambuco, utilizando técnicas de análise de dados e geoprocessamento.

## Tecnologias
- Python
- Pandas (Tratamento de dados)
- Matplotlib & Seaborn (Visualização)
- GeoPandas (Mapas e análise geoespacial)
- Streamlit (Interface do Dashboard)

## O que o projeto faz
- Analisa o perfil dos beneficiários por raça, sexo e PCD.
- Compara a distribuição de bolsas integrais e parciais.
- Identifica os municípios com maior concentração de bolsistas.
- Apresenta um mapa coroplético da distribuição geográfica no estado.

## Como rodar
1. Instale as dependências: pip install -r requirements.txt
2. Execute o app: streamlit run app.py
